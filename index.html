German Medical Dictation → Structured JSON

Transcribe German medical audio with faster-whisper, extract clinical summaries via any OpenAI-compatible LLM, and output validated JSON — all from a single CLI command.

How It Works

Processing Pipeline

Step 1

Audio Input

WAV or MP3 file of German medical dictation passed as CLI argument.

Step 2

Transcription

faster-whisper (CTranslate2) with German language, VAD filtering, beam search.

Step 3

LLM Extraction

Structured prompt sent to Ollama, OpenAI, or any compatible API endpoint.

Output

Structured JSON

Pydantic-validated ClinicalSummary with four standardized fields.

Data Model

ClinicalSummary Schema

patient_complaint

Chief complaint or reason for visit, in the patient's terms.

findings

Clinical findings from examination or history.

diagnosis

Working or confirmed diagnosis.

next_steps

Recommended next steps: treatment, referrals, follow-up.

Why These Choices

        
Design Decisions

      


      

        

          

            
          

          
faster-whisper over openai-whisper

          
~4× faster on CPU via CTranslate2 quantization. Same Whisper model weights, dramatically better inference speed without GPU.

        


        

          

            
          

          
Whisper small default

          
Best accuracy/speed tradeoff for German medical terminology. Easily configurable to tiny (fast) or large-v3 (best quality) via CLI flag.

        


        

          

            
          

          
OpenAI-compatible client

          
Works with Ollama, vLLM, OpenAI, Azure — swap your LLM provider with environment variables alone. Zero code changes.

        


        

          

            
          

          
Pydantic validation

          
LLM output is validated against the ClinicalSummary schema. Malformed or incomplete JSON fails fast with clear, actionable errors.

        


        

          

            
          

          
stderr / stdout split

          
Progress logs go to stderr, JSON result to stdout. Clean for Unix piping: meddict audio.wav | jq.

        


        

          

            
          

          
uv + Docker

          
Fast dependency resolution with uv. Dockerfile pre-downloads the Whisper model for instant cold starts in production.
